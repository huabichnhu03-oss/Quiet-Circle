# Quiet Circle (MoodMind)

Mobile-first mental wellness web app for LGBTQ+ communities.

- **Production:** https://www.quiet-circle.app
- **Vercel preview:** https://quiet-circle-gamma.vercel.app
- **Repo:** https://github.com/huabichnhu03-oss/Quiet-Circle

## Quick start

```bash
npm install
cp .env.example .env   # add Clerk keys
npm run dev            # http://localhost:5000
```

## Docs

- **`CAUTION.md`** — read before every GitHub push (local test keys vs live Vercel keys)
- **`checklist.md`** — agent + owner launch checklists (per-user data, deploy)
